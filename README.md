# Hospital Management System (Mini Project)

This is a MERN stack mini project developed for learning full-stack development.

# Features
- User Authentication (Admin, Doctor, Patient)
- Appointment Booking System
- Blood Bank Management
- Bed Management System

# Tech Stack
Frontend: React.js  
Backend: Node.js, Express.js  
Database: MongoDB  

# Modules
1. Admin Dashboard
2. Doctor Dashboard
3. Patient Dashboard
4. Appointment Management
5. Blood Request System
6. Bed Management


Backend:
cd hospital-backend
npm install
npm start

Frontend:
cd hospital-frontend
npm install
npm start
